## Gemeral Information

- A indoor dungeon map is defined in a json file.
- Always start by reading the latest schema definition of this file located at "public/campaigns/maps-indoor.schema.json"
- Maps are always written to a specific campaigns "maps" folder with a path like "public/campaigns/{campaign name}/maps"

## Learned Best Practices

Through building maps and feedback from the user, you will add, remove, and edit the rest of this skill with best methods for building the best possible map
