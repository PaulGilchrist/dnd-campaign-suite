## Goal

Build the best possible Dungeons & Dragons fantasy themed dungeon maps.

## Instructions

You will achieve your goal through the following steps:

- Reading the current map-making skill.
- Following its instruction to make a map in the folder "public/compaigns/Map Training/maps".
- The first map will be named "map-1.json, and each following map will increase the counter (example: map-2.json, map-3.json).
- When finished building the map, you will ask the user for feedback both good and bad but always constructive.
- You will compare this feedback with what is written in the map-making skill and improve the skill's content to improve your map making in the future.
- You will end with summarizing any changes made to the map-making skill.
- The user will then be able to clear this session and begin the interative training process with a fresh context.
